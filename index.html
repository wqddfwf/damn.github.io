四年级数学公式

1. 数的运算

加法交换律：a + b = b + a

加法结合律：(a + b) + c = a + (b + c)

乘法交换律：a × b = b × a

乘法结合律：(a × b) × c = a × (b × c)

乘法分配律：a × (b + c) = a × b + a × c

2. 面积计算

长方形面积 = 长 × 宽 → S = a × b

正方形面积 = 边长 × 边长 → S = a × a

3. 周长计算

长方形周长 = (长 + 宽) × 2 → C = (a + b) × 2

正方形周长 = 边长 × 4 → C = a × 4

五年级数学公式

1. 小数运算

小数乘法：先按整数乘法算出积，再看因数中一共有几位小数，就从积的右边起数出几位点上小数点

小数除法：除数的小数点向右移动几位，被除数的小数点也向右移动几位(位数不够时补0)，然后按整数除法计算

2. 分数运算

分数加减法：同分母分数相加减，分母不变，分子相加减

分数乘法：分子相乘作分子，分母相乘作分母

分数除法：除以一个数等于乘以这个数的倒数

3. 几何图形

平行四边形面积 = 底 × 高 → S = a × h

三角形面积 = 底 × 高 ÷ 2 → S = a × h ÷ 2

梯形面积 = (上底 + 下底) × 高 ÷ 2 → S = (a + b) × h ÷ 2

4. 体积计算

长方体体积 = 长 × 宽 × 高 → V = a × b × c

正方体体积 = 棱长 × 棱长 × 棱长 → V = a × a × a

六年级数学公式

1. 分数运算

分数加减法：异分母分数先通分，再按同分母分数加减法计算

分数乘分数：分子乘分子，分母乘分母

分数除法：除以一个分数等于乘以它的倒数

2. 百分数

百分数与小数的互换：百分数去掉百分号，小数点向左移动两位 → 小数；小数小数点向右移动两位，加上百分号 → 百分数

百分数与分数的互换：百分数写成分母是100的分数，再约分 → 分数；分数先化成小数，再化成百分数

3. 比例

比例的基本性质：a : b = c : d → a × d = b × c

正比例关系：y/x = k (k为常数) → y = kx

反比例关系：x × y = k (k为常数) → y = k/x

4. 圆的公式

圆的周长 = π × 直径 = 2 × π × 半径 → C = πd = 2πr

圆的面积 = π × 半径² → S = πr²

5. 圆柱和圆锥

圆柱侧面积 = 底面周长 × 高 → S侧 = Ch = 2πrh

圆柱表面积 = 侧面积 + 2 × 底面积 → S表 = 2πrh + 2πr²

圆柱体积 = 底面积 × 高 → V = πr²h

圆锥体积 = 1/3 × 底面积 × 高 → V = 1/3πr²h
